What is this?
This site will scrape RSS feeds and download files to specified directory. Intended to be run as a cronjob there
is a manual mechanism for executing watch lists in the admin section.
Features:
  • Not supported on windows platforms
  • Scrape multiple RSS Feeds
  • Download multiple filetypes (NZB, TORRENT, ZIP, etc)*
  • Authentication w/saved cookies (through wget)
  • Build a watch list to monitor keywords in RSS feeds
  • Dictate watch list items and rss feed types (TV, Movie, Music etc)
  • Build A library of Items from directories on your system
  • REGEX match Season and Episode information in filename
  • SQLite 2.1 database driven
  • Maintain tables through interface
*When downloaded file information is stored and checked before downloading the same file again
